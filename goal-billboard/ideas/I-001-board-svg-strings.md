# I-001 — Board relationship "strings" (the literal SVG link layer)

Render the goal→plan→audit→research relationships as actual corkboard string —
an SVG node-link layer over the Board, reusing the Blueprint WIRING-MAP's
`<path>` + end-dot technique, default-off with focus-reveal (focus a card → only
its threads light, neighbors dim). The Board ships today with link *chips* and
by-goal grouping; the literal strings are the careful follow-up that was
deliberately held out of the core pass (hairball risk at 85 audits → G2 needs
the focus-reveal + active-only-default discipline first). Captured for the user's
review — not greenlit.
