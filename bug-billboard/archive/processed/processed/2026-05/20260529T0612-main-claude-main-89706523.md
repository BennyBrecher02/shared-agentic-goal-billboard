## Description
Product modal's "Request a quote" is a static <a href="/contact"> — identical for all 16 chargers; the open handler never rewrites href. Opening any unit → generic /contact, no product attached, no pre-fill. Page copy promises "Pick the units. We'll price the deploy." data-product={p.id} already on the card → ?product={id} + contact pre-fill is the fix. Found independently by both forms-UX + products-UX audits.
