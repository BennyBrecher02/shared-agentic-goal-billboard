## Fix attempted
B-K7-b Phase A plow BG (2026-05-27, ~8 min wall-clock):
- Issue 1 (affordance): 1.5px subpixel-rounds to 1px on standard-DPI; integer 2px + darker neutral #b9c1cc clears the affordance gap at all DPIs.
- Issues 2 & 3 shared root cause: grid `align-items: normal` (= stretch) was over-extending the aside track while form-card padding-top (~52px) ≠ aside padding-top (~40px). - Mobile responsive untouched: `@media (max-width: 991px) { .form-wrap > aside { padding-top: 0; } }` restores tight mobile stacking.
- Diff zones touched: `.form-wrap` / `.form-wrap > aside` / `.field input` blocks in app.css only. Zero changes to src/pages/contact.astro. No sibling-BG zone crossing.
